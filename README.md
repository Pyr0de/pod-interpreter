# Pod Interpreter

```
func add(a, b) {
    return a + b;
}
// Add 2 arrays
init x = 10;
init y = 12.2;
x = 2 ^ 2; // 4

init sum = add(x, y); // 4 + 12.2 = 16.2

print(sum)
```

### Operators
`+`, `-`, `*`, `/`, `^`, `%`, `++`, `--`, `==`, `!=`, `>`, `>=`, `<`, `<=`, `&&`, `||`
### Function
`func`, `return`
### Initialize
`init`
### If and loops
`if`, `else`, `for`, `while`
### Data types
`string`, `number`, `true/false`
### Other
`;`, `(`, `)`, `{`, `}`
